Module Origami::Object
In: origami/object.rb
origami/obfuscation.rb

Parent module representing a PDF Object. PDF specification declares a set of primitive object types :

  • Null
  • Boolean
  • Integer
  • Real
  • Name
  • String
  • Array
  • Dictionary
  • Stream
